🛳️ 船舶 AIS 軌跡比對 — 點到點線段

每艘船兩家分別的航行軌跡：紅線 = 船訊網、綠線 = SDataPro。若綠線完整覆蓋紅線 → 代表 SDataPro 的 AIS 資料能取代船訊網。
🛳️ 圖表說明｜AIS 軌跡比對 看什麼：該船在觀測期間，兩家分別記錄到的航跡點以時間順序連線。 欄位定義：
  • 紅線 = 船訊網軌跡（依 xy_lasttime_fd 連線）
  • 綠線 = SDataPro 軌跡（依 sd_timestamp 連線）
  • 每段最多顯示 200 個取樣點（原始點數列在圖例與右上角資訊）
  • 頂部按鈕可切換顯示／隱藏單一資料源，方便比對
▸ 怎麼讀：
  • 兩條線完全重疊 → 兩家航跡一致 → SDataPro 可取代
  • 綠線比紅線更長／更密集 → SDataPro 收得比船訊網多，肯定可取代
  • 綠線有但紅線沒有的區段 → SDataPro 額外覆蓋了船訊網漏掉的時段
  • 只有紅線沒綠線的區段 → SDataPro 在那段時間漏掉資料（少見）
▸ 結論：多數船兩條線高度重疊；少數 cross-Pacific 或杜拜異常船會看到部分區段的差異，多源於 AIS 接收條件，非 SDataPro 演算法問題。

① 資料收集密度

兩家在相同時段內，誰收到的 AIS 紀錄比較多？更新頻率有沒有差異？
📊 圖表說明｜兩家總更新次數對比 看什麼：7 天內，兩家分別收到了多少筆獨立的 AIS 更新。 欄位定義：用 unique timestamps 計算（避免重複比對紀錄被重複計算）。 ▸ 怎麼讀：柱越高代表收到的更新越密。 ▸ 結論：SDataPro 共 66,077 次 vs 船訊網 63,774 次，SDataPro 多 3.6%，密度略勝。
📊 圖表說明｜資料更新週期分布 看什麼：兩家收到下一筆資料的「等待時間」分布。 欄位定義：橫軸為更新週期（分鐘），縱軸為紀錄數，越往左 = 越頻繁。 ▸ 怎麼讀：分布峰值越靠近 0 代表更新越頻繁。 ▸ 結論：SDataPro 中位週期 4 分、船訊網 5 分，SDataPro 更新略快。
📊 圖表說明｜各船每小時平均紀錄數 看什麼：每艘船在觀測期間，兩家平均每小時收到幾筆資料。 欄位定義：橫條長度 = 平均每小時紀錄數；綠 = SDataPro，紅 = 船訊網。 ▸ 怎麼讀：同一艘船兩條柱中，較長者代表該家收得較密；綠長 = SDataPro 在這艘船上更密集。 ▸ 結論：大多數船 SDataPro 與船訊網的密度相當；少數船 SDataPro 顯著更密集。

② SDataPro 準確度（以船訊網為基準）

把船訊網當作參考值，SDataPro 與它的差距就是 SDataPro 的「誤差」。
📊 圖表說明｜SDataPro 位置誤差分布 看什麼：SDataPro 與船訊網「位置差距」的直方圖，最大顯示 20 km。 欄位定義：橫軸 = 位置誤差 (km)，縱軸 = 紀錄數。 ▸ 怎麼讀：分布越集中於 0 附近代表 SDataPro 越準。 ▸ 結論：絕大多數紀錄誤差 < 2 km，超過 5 km 占比極低。
📊 圖表說明｜SDataPro 艏向誤差分布 看什麼：SDataPro 與船訊網「艏向 hdg 差距」的直方圖。 欄位定義：橫軸 = 艏向誤差（度），縱軸 = 紀錄數。 ▸ 怎麼讀：越往左集中代表越準。 ▸ 結論：多數紀錄誤差 ≤ 5°，超過 10° 為少數。
📊 圖表說明｜SDataPro 船速誤差分布 看什麼：SDataPro 與船訊網「船速 sog 差距」的直方圖。 欄位定義：橫軸 = 船速誤差（節），縱軸 = 紀錄數。 ▸ 怎麼讀：越往左集中代表越準。 ▸ 結論：幾乎所有紀錄誤差 < 1 節，船速量測極為一致。

📊 圖表說明｜符合度檢查矩陣 (筆數合格率) 看什麼：每艘船在 4 個指標上「有多少 % 紀錄達門檻」，加上「4 項同時達標」總覽欄。 欄位定義：每格顏色與數字 = 該船該指標的紀錄合格百分比；色階 紅 (低) → 黃 (中) → 綠 (高)，越深綠越好。船舶依「4 項全合格率」排序。 ▸ 怎麼讀：橫看一艘船：每項合格率高低；縱看一個指標：船隊整體通過情況。最右欄是該船 4 項同時達標的紀錄占比，是最嚴格的綜合指標。 ▸ 結論：整體 4 項全合格率動態載入；少數低分船多為 AIS 接收不穩或杜拜異常，並非 SDataPro 演算法問題。
📊 圖表說明｜各船 SDataPro 綜合準確度分數 看什麼：每艘船的綜合準確度分數（依位置 40%、艏向 20%、航向 20%、船速 20% 加權，越高越準）。 欄位定義：橫條長度 = 綜合分數 (0~100)；色階綠→黃→紅 = 準確度高→低。 ▸ 怎麼讀：排在最上方的船 SDataPro 表現最接近船訊網；最下方則差距大。 ▸ 結論：多數船分數 ≥ 70 分；少數低分船多為已知異常船 (杜拜、AIS 訊號不穩)。
