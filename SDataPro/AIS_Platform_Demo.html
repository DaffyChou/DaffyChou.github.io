жёҜеҸЈж“ҒеЎһжҢҮж•ё (Congestion Index, CI) жҳҜдҫқж“ҡ жӯ·еҸІ AIS и»Ңи·Ў иҲҮ еҚіжҷӮи§Җжё¬ иЁҲз®—зҡ„ 0вҖ“1 йҮҸеҢ–еҲҶж•ёгҖӮи¶ҠжҺҘиҝ‘ 1.0 д»ЈиЎЁи¶Ҡж“Ғж“ гҖӮ
CI_anchor В· йҢЁең°ж“ҒеЎһ
CI_anchor = 0.6 Г— (N_now / N_cap) + 0.4 Г— (N_now / N_base)
N_nowпјҡз•¶еүҚйҢЁең°иҲ№ж•ёпјҲдҫқ AIS SOG < 1kn жҢҒзәҢ > 10min еҲӨе®ҡпјү
N_capпјҡйҢЁең°жңҖеӨ§е®№йҮҸпјҲз”ұ DBSCAN иҒҡйЎһжёҜеӢҷе…¬е‘ҠжҺЁдј°пјү
N_baseпјҡжӯ·еҸІеҗҢжҷӮж®өдёӯдҪҚж•ё
CI_flow В· йҖІжёҜжөҒйҮҸ
CI_flow = F_now / F_base
F_nowпјҡйҒҺеҺ» 60 еҲҶйҗҳйҖҡйҒҺйҖІжёҜиө°е»Ҡзҡ„иҲ№иҲ¶ж•ёйҮҸ
F_baseпјҡжӯ·еҸІеҗҢжҳҹжңҹеҗҢжҷӮж®өзҡ„дёӯдҪҚж•ёйҖҡйҒҺйҮҸ
> 1.0 иЎЁзӨәжҜ”е№іеёёз№Ғеҝҷпјӣ< 1.0 иЎЁзӨәжөҒйҮҸијғдҪҺ
CI_berth В· жіҠдҪҚеҲ©з”ЁзҺҮ
CI_berth = B_now / B_cap
B_nowпјҡз•¶еүҚдҪңжҘӯдёӯжіҠдҪҚж•ёйҮҸпјҲиҲ№иҲ¶ SOG < 0.3kn дё”еңЁжіҠеҚҖеӨҡйӮҠеҪўе…§пјү
B_capпјҡжёҜеҸЈзёҪжіҠдҪҚж•ёйҮҸ
жҺҘиҝ‘ 1.0 иЎЁзӨәжіҠдҪҚе№ҫд№Һж»ҝијүпјҢж–°иҲ№иҲ¶е°ҮйӣЈд»ҘеҚіжҷӮеҲҶй…Қйқ жіҠ
CI_port В· з¶ңеҗҲж“ҒеЎһжҢҮж•ё
CI_port = 0.5 Г— CI_anchor + 0.3 Г— CI_flow + 0.2 Г— CI_berth
