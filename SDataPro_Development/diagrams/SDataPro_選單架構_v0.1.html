📊 船隊監控 系統首頁

群組：即時監看
50 艘自家船的全局俯瞰視角：每艘船一行，含 9 維度 dot 矩陣 + 整體風險等級 + 船舶狀態。
每日上班的第一站，目的是「快速掃出哪幾艘船有狀況」，然後點船進入「周圍態勢」深入查看。
沿用 Demo
沿用 voyage-alert-rwd.html 的 #/fleet，依 50 船擴增列表處理（sticky / 虛擬滾動 / 篩選）

🛰 周圍態勢 合併 4 Demo

群組：即時監看
單船的多視角綜合介面。從船隊監控點任一船進入；地圖中央顯示該船 + 5 NM 圓 + 周邊他船。
右上有圖層 toggle：CPA 警示 / 歷史軌跡 / 計畫航線 / 風險區 / AIS 品質——使用者自己決定要看什麼。
右側 panel 列出周邊船清單，點任一艘可在 panel 內展開 AIS 完整性 / 立案告警，不離頁面。
合併以下 Demo
Demo 1 週邊態勢 底層原始視角
Demo 3 CPA ＝ CPA 警示圖層
Demo 2 歷史回放 ＝ 時間軸模式（subroute）
Demo 7 AIS 完整性 ＝ 品質檢視圖層 / 點船展開

⚓ 港口擁塞看板 跨船 / 跨港聚合

群組：港口營運
把多艘自家船的周圍監控資料按港口聚合，看每個港的擁塞程度。
145 重點商港總覽 + 篩選 + 全球熱點地圖；點任一港口進入單港口頁面：港區海圖 + CI 指數 + 錨地 / 走廊 / 泊區 + 自家船 ETA 清單 + 外來船觀測。
對應 Demo
Demo 4 港口擁塞 完整保留，含單港口詳情

📋 港口情資 SitRep 含 ETA 動態

群組：港口營運
以「自家船 → 目的港」為主體的情資彙整報告。距港 24 / 12 / 6 hr 自動觸發 SitRep。
頁面內部分頁：動態 ETA（T1+T2+T3+T4 預測 + P25/P50/P75）/ SitRep 報告（在場船 + Demurrage 風險 + 航速建議）。
合併以下 Demo
Demo 5 ETA 預測 ＝ 子分頁 1「動態 ETA」
Demo 6 SitRep ＝ 子分頁 2「情資報告」

⚠ 事件處理 既有強化

群組：事件管理
9 維度告警自動立案 + 操作員流程（New → Ack → In Progress → Resolved → Closed）。
Modal 內含「查歷史回放」「查 AIS 品質」按鈕——但不會跳到別的 sidebar 選單，是用 modal 內 panel 展開或開新 tab 子視窗。
沿用
沿用 voyage-alert-rwd.html 的 #/events，依 50 船 × 9 維度 = 450 監控點重新設計聚合策略

🕐 警示時間線 既有保留

群組：事件管理
混合 Warning Mail + open_event 的時間軸 DESC 列表。
提供「最近 24/72 hr 警示一覽」視角，與事件處理頁互補（事件處理頁是「待辦」視角，時間線是「歷史」視角）。
沿用
沿用 voyage-alert-rwd.html 的 #/timeline

🗺 計畫版本管理 新功能

群組：航行計畫
每艘船的多版本航行計畫管理：Active / Superseded / Archived；新版來源（StormGeo / WNI）+ 差異比對地圖。
是 M5 模組的核心，補上既有 Demo 缺的「版本歷史」功能。
對應
全新功能；對應 Fleet War Room §5.2 「航行計畫資料版本管理」

💡 智慧航線建議 Phase A 起步

群組：航行計畫
基於 1–2 年歷史 AIS 自動推薦航線（同港口對、同季節、平均速度、常見偏移區）。
Phase A 只做歷史推薦；Phase B 加氣象；Phase C 動態優化。
對應
全新功能；對應戰情室任務說明 §5「智慧航行計畫」

⚙ 閾值設定 既有強化

群組：系統管理
6 維度的 Warning / Critical 閾值調整，Save 後立即重算 9 維度告警。
加入 CPA / AIS 完整性 / 港口擁塞 的閾值（新增約 9 個維度）。
沿用 + 擴增
沿用 voyage-alert-rwd.html 的 #/thresholds，新增 CPA / AIS / 港口擁塞 閾值

📄 稽核紀錄 新功能

群組：系統管理
全系統 audit log：actor + role + before/after + comment + timestamp。
除了事件處理，CPA 確認 / 閾值變更 / 計畫版本切換 / 採納降速建議 都會寫入。
對應
全新功能；對應 Fleet War Room §4.3「稽核紀錄」

⬇ 報表匯出 新功能

群組：系統管理
事件統計 / CPA 熱點 / ETA 準確率 / 港口擁塞趨勢 等月度報表。
匯出 Excel / PDF，給管理層、內外稽、客戶檢討。
對應
全新功能；月度循環的最後一站
