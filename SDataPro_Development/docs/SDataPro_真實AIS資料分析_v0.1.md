## 3. 資料欄位結構

CSV 共 14 欄：

| 欄位 | 內容 | 對應 SDataPro Demo |
|---|---|---|
| `source_id` | 收集器 ID（= 自家船） | vessel_id（用化名對應）|
| `message_type` | AIS Msg Type 1/3/5/18 等 | 區分動態 / 靜態 / Class B |
| `mmsi` | 目標 MMSI | 自船或周邊船 |
| `target_category` | vessel / unknown / base_station | 區分船舶 / 浮標 |
| `timestamp` | DD/M/YYYY HH:MM:SS | UTC → 轉 UTC+8 |
| `lat`, `lon` | 緯度 / 經度 | 地圖座標 |
| `course` | COG | 航向 |
| `speed` | SOG | 航速 |
| `turn` | ROT（轉向率）| 不在 Demo |
| `heading` | 真航向 | 船舶旋轉角度 |
| `navigation_status` | 0=航行中 / 1=錨地 / 5=靠泊 等 | 船舶狀態 |
| `aid_type` | 浮標類型 | 非船舶 marker |
| `epfd` | GPS 來源 | 不在 Demo |

## 4. 預處理 Pipeline 設計建議

### 4.1 目標

把 1.2 GB CSV → ~50 MB 的 JSON 集合，分散到多個檔案，瀏覽器按需 load。

### 4.2 預處理腳本架構

```
preprocess/
├── 01_clean.py            # 過濾異常 lat/lon、合併 src 13+50、處理 MMSI 4132203
├── 02_identify_own.py     # 確認每個 source 的自船 MMSI + 軌跡
├── 03_extract_static.py   # 解析 type 5/19/24 得船名 / IMO（化名前先存真名 → 對應化名）
├── 04_downsample.py       # 軌跡 downsample（每 5 分鐘 1 點）
├── 05_surrounding.py      # 每艘自船的周邊觀測（< 5 NM 過濾、15 分鐘 snapshot）
├── 06_anonymize.py        # 套用化名規則：MMSI / 船名 → 化名
└── 07_generate_demo.py    # 輸出 Demo 用 JSON
```

### 4.3 預期輸出檔案結構

```
demo/data/
├── vessels.json                          # 20 自家船基本資料（化名 + 國籍 + 觀測中心）
│                                         # ~15 KB
├── tracks/
│   ├── ship_yy_01.json                   # 遠洋探索號 8 天軌跡（每 5 分鐘 1 點 = ~2300 點）
│   ├── ship_yy_02.json
│   └── ... 20 個檔案，每個 ~200 KB
│                                         # 合計 ~4 MB
├── surroundings/
│   ├── ship_yy_01_20260527.json          # 該船該日的周邊觀測 snapshot
│   ├── ship_yy_01_20260528.json
│   └── ... 20 船 × 8 天 = 160 個檔案
│                                         # 合計 ~30 MB
├── events.json                           # 自動生成的 CPA / Route 事件
│                                         # ~50 KB
└── ports.json                            # 145 港口 + 觀測到的進出港統計
                                          # ~50 KB
```

### 4.4 化名對應原則

| 真實 MMSI | 化名 MMSI | 化名船名 |
|---|---|---|
| 416466000（台灣）| 416001001 | 遠洋探索號 / YUAN YANG EXPLORER |
| 416465000（台灣）| 416001002 | 遠洋啟航號 / YUAN YANG VOYAGE |
| ... | ... | ... |

化名規則：
- 保留國籍前綴（416 → 416001001~，636 → 416002001~）
- 周邊船 MMSI 也化名（用 hash 一致映射，同 MMSI 永遠對應同化名）
- 浮標 / base station 化名為 `KH-BUOY-XX`、`KH-BASE-XX`
