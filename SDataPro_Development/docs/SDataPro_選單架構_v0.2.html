📊 船隊監控 系統首頁

群組：即時監看 · 跨船全局視角
50 艘自家船的全局俯瞰：每艘船一行，9 維度 dot 矩陣 + 整體風險 + 船舶狀態 + 動態 ETA + AIS 品質 chip。
每日上班第一站，目的是「快速掃出哪幾艘船有狀況」，然後點船進入「船舶詳情」深入查看。
沿用
沿用 voyage-alert-rwd.html 的 #/fleet，依 50 船擴增（sticky / 虛擬滾動 / 篩選）

🚢 船舶詳情 關鍵整合

群組：即時監看 · 單一船舶所有視角
從船隊監控點任一船進入，或在 sidebar 切後預設打開最近檢視的船。
