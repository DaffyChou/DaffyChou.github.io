1. 總運力與能耗趨勢 連動篩選

EEOI 計算公式、業界基準與燃油涵蓋範圍：
公式：EEOI = Σ 耗油量 (MT) × CF / Σ (貨運量 ton × 航距 nm)
CF 係數：VLSFO / HFO ≈ 3.114 g CO₂/g、LSMGO / MDO ≈ 3.179 g CO₂/g、LNG ≈ 2.750 g CO₂/g
數值越低代表單位運輸工作碳排強度越小、能效越佳
本圖燃油範圍：統計所有 BDN 紀錄之燃油耗量加總（HFO / VLSFO / LSMGO / MDO / LNG 等），不限類型。各燃油類型之結構拆分請見「圖表 4：燃油消耗結構統計」。

判讀指南

藍色柱狀圖為各期間貨運周轉量 (M Ton-miles)，橘色折線為船隊總油耗 (MT)，綠色折線為 EEOI 碳強度。兩線脫鉤代表運能成長但油耗受控。點擊任一資料點可展開單船能耗細部資訊。

診斷觀察

當前區間貨運周轉量穩定成長，總油耗保持平穩，EEOI 呈緩降趨勢。點擊圖表任一時間點可進一步識別各船 EEOI 排名，紅色標示為高於船隊平均、拖低整體效率之關鍵船舶。

2. 船隊營運狀態能耗佔比 連動篩選

佔比 (%) 計算說明與突波解讀：
Waste % = (區間空等總耗油 / 船隊總耗油) × 100%。
若逢淡季(總油耗分母小)，或有耗能大船發生塞港(空等分子大)，皆會產生佔比突波。

判讀指南

最上方紅色區塊代表在港外空等時消耗的浪費。佔比激增代表塞港或排程延宕。

診斷觀察

請注意圖表中 紅色區塊凸起 的區間。建議點擊該柱狀圖紅區調閱明細追查責任。

防偏誤聲明 (Baseline Correction)：
請透過右上角下拉選單篩選同級距別之船舶進行比較。不同噸位船舶之推進基準不同，切勿跨級比較以免產生判讀誤差。

判讀指南

  • 中位數 (紅線): 代表 50% 效率在此值以上，常態體質基準。
  • 變異度 IQR (藍箱): 箱體越短代表操作表現越穩定。
  • 離群值 (圓點): 非常規運行或異常數據點，需優先追查。

診斷觀察

船隊操船穩定度落差明顯。成功輪/創新輪可作標準SOP；而永續輪/堅韌輪需介入改善。

4. 船隊燃油消耗總量與結構統計 連動篩選

燃油結構分析：
區間總耗油量 = 常規重油 VLSFO (主機使用) + 昂貴輕油 LSMGO (發電機使用)

判讀指南

柱狀圖堆疊顯示船隊在各區間的重油(VLSFO)與輕油(LSMGO)耗量，折線圖對應當期總耗油量。

診斷觀察

請密切關注紅色的 輕油(LSMGO) 佔比變化。若輕油總量異常攀升，通常代表船隊港內待命時間拉長，需留意高昂的發電成本。

5. 加油港口排行 連動篩選

採購集中度評估：
統計該區間內，各港口之燃油交付單(BDN)總量 (MT)。

判讀指南

顯示前五大加油港口與加注總噸數，幫助採購單位檢視燃油合約分佈狀態。

診斷觀察

透過掌握各港口的加注噸數集中度，採購單位即可憑藉大宗採購的規模優勢 (Economies of Scale)，向當地供應商爭取更好的長約折扣。
