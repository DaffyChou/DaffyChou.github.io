# 摯陞 SRP 船隊戰情室系統 — 警報 Email 範本

本文件對應 [alert_demo.html](alert_demo.html) 中 4 類共 13 種警報的 email 通知範本。
所有 `{{變數}}` 為佔位符，由警報系統觸發時自動代入。

---

## 一、航次警報（Voyage Alerts）

### 1. 進／出港狀態（gate）

```
主旨：【航次警報】{{shipName}} 已{{dir}}地理圍欄 — {{geofenceName}}

收件人您好，

船舶「{{shipName}}」（IMO：{{imo}}）已於 {{eventTimeUTC}} (UTC) {{dir}}預設地理圍欄
「{{geofenceName}}」。

▎事件資訊
  · 位置：Lat {{lat}} / Lon {{lon}}
  · SOG：{{sog}} kn   Heading：{{heading}}°
  · 預計下一港：{{nextPort}}（ETA {{eta}}）

▎建議動作
  · 確認 Port Agent 已收到 NOR 通知
  · 檢查靠泊／離泊作業排程

— 摯陞 SRP 船隊戰情室系統
```

### 2. 抵港時間 ETA（eta）

```
主旨：【航次警報】{{shipName}} ETA {{ba}} {{n}} 小時提醒

船舶「{{shipName}}」目前距預估抵達 {{destPort}} 尚有 {{hoursToEta}} 小時，
已進入「ETA {{ba}} {{n}} 小時」之警示區間。

▎航次資訊
  · 目的港：{{destPort}}
  · 原訂 ETA：{{plannedEta}}
  · 最新 ETA：{{currentEta}}（變動 {{delta}} 小時）
  · 目前位置：Lat {{lat}} / Lon {{lon}}，SOG {{sog}} kn

▎建議動作
  · 與 Port Agent 再次確認泊位安排
  · 通知貨主／傭船人最新 ETA

本警報每 1 小時提醒一次，限制 3 次。
— 摯陞 SRP 船隊戰情室系統
```

### 3. 航線偏移（off）

```
主旨：【航次警報】{{shipName}} 偏離航線 {{deviation}} 海浬

船舶「{{shipName}}」目前位置已偏離預計航線 {{deviation}} 海浬
（門檻 {{threshold}} 海浬），請即時確認。

▎位置資訊
  · 目前位置：Lat {{lat}} / Lon {{lon}}
  · 預計航點：Lat {{plannedLat}} / Lon {{plannedLon}}
  · COG：{{cog}}°   SOG：{{sog}} kn
  · 偏離時間：自 {{deviationStart}} 起持續 {{durationMin}} 分鐘

▎建議動作
  · 與船長確認偏航原因（避風／避碰／海流？）
  · 評估燃油與 ETA 影響

本警報每 30 分鐘提醒一次，直到船舶回到航線。
— 摯陞 SRP 船隊戰情室系統
```

### 4. 惡劣海況預警（sea）

```
主旨：【航次警報】{{shipName}} 前方海域氣象惡劣

船舶「{{shipName}}」未來 {{forecastHours}} 小時內航經海域，
氣象條件已達警示門檻。

▎氣象資訊
  · 預測浪高：{{wave}} m（門檻 > {{waveLimit}} m）
  · 預測風力：{{wind}} 級（門檻 > {{windLimit}} 級）
  · 預測時間／位置：{{forecastTime}} @ Lat {{lat}} / Lon {{lon}}
  · 資料來源：{{weatherSource}}

▎建議動作
  · 通知船長加強瞭望與固縛
  · 評估減速、繞航或避風方案
  · 提前向傭船人／貨主預警可能延誤

條件持續存在時將再次發送。
— 摯陞 SRP 船隊戰情室系統
```

### 5. 船速異常下降（spd）

```
主旨：【航次警報】{{shipName}} 船速異常下降 {{dropPct}}%

船舶「{{shipName}}」主機 RPM 維持正常，但對地速度（SOG）較預期下降
{{dropPct}}%（門檻 {{threshold}}%），疑似船體阻力異常。

▎運轉資訊
  · 預期 SOG：{{expectedSog}} kn
  · 實際 SOG：{{actualSog}} kn
  · 主機 RPM：{{rpm}}（變動 {{rpmDelta}}%）
  · 海流／風向：{{currentInfo}}

▎可能原因
  · 船底海生物附著
  · 異常海流／逆風
  · 主機效能下降

▎建議動作
  · 安排下次入塢清底評估
  · 比對近 7 天能耗趨勢

本警報每 4 小時提醒一次，限制 3 次。
— 摯陞 SRP 船隊戰情室系統
```

### 6. 限制區域進入提醒（zone）

```
主旨：【航次警報】{{shipName}} 即將進入 {{areaName}} — 距邊界 {{distance}} 海浬

船舶「{{shipName}}」距 {{areaName}} 邊界僅 {{distance}} 海浬
（門檻 < {{threshold}} 海浬），請確認進入該區域所需準備事項。

▎航行資訊
  · 目前位置：Lat {{lat}} / Lon {{lon}}
  · 航向／速度：{{cog}}° / {{sog}} kn
  · 預估抵達區域：{{etaArea}}
  · 目標區域：{{areaName}}

▎區域進入準備清單
  · ECA：切換至 ≤ {{sLimit}}% 低硫油、Oil Record Book 紀錄
  · HRA：部署反海盜措施（PCASP、Citadel 演練、加速通過）
  · JWLA 戰爭風險區：加保 War Risk 保險、評估替代航線

說明：本警報為位置型提醒。因油耗資料由船端午報（T+1）提供，
系統無法即時驗證實際油料切換動作；合規執行須由船端 Oil Record Book、
午報資料事後核對。

本警報每 30 分鐘提醒一次。
— 摯陞 SRP 船隊戰情室系統
```

▎本月達標清單

## 三、Bunker 管理警報（Bunker Management）

### 9. 最低庫存警示（low）

```
主旨：【Bunker】{{shipName}} 油料 ROB 已低於安全庫存

船舶「{{shipName}}」目前剩餘油量（ROB）已低於系統推估之動態安全庫存，
請儘速規劃補油。

▎庫存資訊
  · 油種：{{fuelType}}
  · 目前 ROB：{{rob}} MT
  · 動態安全庫存：{{safetyStock}} MT
  · 預估可航行：{{daysRemaining}} 天 / {{nmRemaining}} nm

▎建議動作
  · 與 Bunker Trader 確認下次補油量
  · 評估是否需就近補油

本警報每日提醒 1 次，直到 ROB 高於安全庫存。
— 摯陞 SRP 船隊戰情室系統
```

### 10. 油耗效能偏差（high）

```
主旨：【Bunker 月報】{{month}} 月 — {{count}} 艘船舶油耗效能偏差

下列船舶上月實際油耗已超過本船性能曲線預期（門檻：> {{n}}%）。
基準採 SOG-RPM-Draft 性能曲線並依 ISO 19030 對風浪做標準化修正，
已排除航線與海況差異；偏差再拆解為 Hull / Propeller / Engine
三項貢獻度，直接指向排查方向。

▎效能偏差清單
┌──────────────┬───────────┬──────────┬──────────┬──────────────────┐
│  船舶         │ 實際 g/kWh│ 曲線預期 │  偏差 %  │  主要貢獻         │
├──────────────┼───────────┼──────────┼──────────┼──────────────────┤
│  {{ship1}}    │ {{c1}}    │ {{e1}}   │ +{{p1}}% │ {{src1}}         │
│  {{ship2}}    │ {{c2}}    │ {{e2}}   │ +{{p2}}% │ {{src2}}         │
└──────────────┴───────────┴──────────┴──────────┴──────────────────┘

▎建議排查項目（依貢獻度指向）
  · Hull 偏差為主：安排下次入塢、船底清洗評估
  · Propeller 偏差為主：考慮螺槳清磨或拋光（polishing）
  · Engine 偏差為主：燃油噴射、增壓器、缸壓量測診斷
  · 持續監測：對照近 6 個月 baseline 確認趨勢

完整分析：{{reportLink}}
— 摯陞 SRP 船隊戰情室系統
```
